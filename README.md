# Table-merge
#### 用途：合并多个Excel表格
#### 支持的表格格式：XLSX、LSX、CSV格式均可。
#### 合并表格便捷版能实现的功能：a.提取表格名并新增在合并后的第一列。b.即使各表格标题长度不一样，相同字段顺序不完全相同，合并时相同字段也会自动对齐。
